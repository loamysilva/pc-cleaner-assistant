PC Cleaner Assistant

Funções:
- Scan de arquivos grandes
- Limpeza segura
- Filtro por risco
- Exportação de relatório
- Exclusão segura para lixeira

Como usar:
1. Abra o programa
2. Clique em "Scan rápido"
3. Revise os arquivos encontrados
4. Limpe apenas arquivos SEGURO

Desenvolvido em Python + Tkinter
