# PC Cleaner Assistant

Assistente de limpeza inteligente para Windows.

## Funções

- Scan rápido de arquivos pesados
- Filtro por risco
- Limpeza segura
- Limpeza de arquivos selecionados
- Exportação de relatório
- Interface moderna em Tkinter

## Tecnologias

- Python
- Tkinter
- Send2Trash
- PyInstaller

## Como executar

```bash
py app.py
```

## Executável

O executável pode ser encontrado na pasta:

```bash
dist/
```
